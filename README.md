# 🏠 Deep Learning for Real Estate Price Prediction

## 📖 Overview
This project leverages both traditional machine learning and deep learning techniques to predict real estate prices. The workflow encompasses the entire data science pipeline, from importing and exploring property datasets to visualizing key market trends, preparing the data, and ultimately training and comparing a Linear Regression model with an Artificial Neural Network (ANN).

## 🎯 Objective
The primary objective is to build a robust predictive model capable of accurately estimating property values based on various housing features. By establishing a baseline with Linear Regression and then implementing a deep learning ANN, the project aims to evaluate and identify the most effective algorithm for continuous price prediction in the real estate market.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Deep Learning:** TensorFlow / Keras (ANN)
* **Machine Learning:** Scikit-Learn (Linear Regression)
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the business case and defining the objective for real estate valuation.
* **Exploratory Data Analysis (EDA):** Investigating the property dataset to find patterns, missing values, and outliers.
* **Data Visualization:** Creating insightful plots to understand the distribution of prices and feature correlations.
* **Data Preparation:** Cleaning, scaling, and formatting the data specifically for neural network training.
* **Baseline Modeling:** Training and evaluating a traditional Linear Regression algorithm for price prediction.
* **Deep Learning (ANN):** Architecting, training, and rigorously evaluating an Artificial Neural Network to predict continuous real estate values.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Deep-Learning-For-Real-Estate-Price-Prediction.git

2. Navigate into the downloaded project folder:
   cd Deep-Learning-For-Real-Estate-Price-Prediction

3. Install the required dependencies:
   pip install pandas numpy scikit-learn tensorflow matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to run the analysis and train the models.
